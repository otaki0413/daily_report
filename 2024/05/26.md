  - python の time モジュールで sleep 処理の実装や現在時刻の構造データ取得ができる
  - モジュールのトップレベルで関数をコールしないほうがよい
  - 正規表現いろいろ
  - デコレータを使って、関数の実行時間を計測できる（下記参考）

    ```
    # デコレータ
    def timer(func):
        def inner(*args, **kwargs):
            before = time.time()
            func(*args, **kwargs)
            after = time.time()
            print(f"{func.__name__} took {after-before:.2f} sec")
        return inner

    # 使い方
    @timer
    def hoge():
      ....
    ```
